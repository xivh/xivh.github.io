Blind for Disrupt the District

I used Python to perform natural language processing and regex on a pdf resume. When names or phone numbers were matched, they were removed from the resume. This project won 3rd place at Disrupt the District and a finalist spot for the UVA Entrepreneurship Cup Concept Competition. (Github)

Google Cloud and NCAA Machine Learning Competition

I used Python to do machine learning on a data set of NCAA teams. After cleaning the data in Excel, I did my best to create a model using scikit-learn. I succesfully predicted many of the first round games, including UVA's historic loss, but my model wasn't that good in the later rounds... (Github)
